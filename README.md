## 新系統常用的套件/程式

### 1. Nvidia driver [link](https://launchpad.net/~graphics-drivers/+archive/ubuntu/ppa)

```
1. sudo add-apt-repository ppa:graphics-drivers/ppa
2. sudo apt-get update
3. 之後進入System settings -> Software & Updates
4. 選擇最新版的Nvidia驅動程式
```
